# ⚡ دليل البدء السريع

## 🚀 ابدأ خلال 3 دقائق

---

### الخطوة 1: التثبيت (دقيقة واحدة)

```bash
# انتقل إلى المجلد
cd SEC_SYSTEM3_ENHANCED

# ثبّت المكتبات المطلوبة
pip install -r requirements.txt
```

أو بدون ملف requirements:
```bash
pip install requests pandas openpyxl yfinance matplotlib
```

---

### الخطوة 2: التشغيل (10 ثوانٍ)

```bash
python main.py
```

---

### الخطوة 3: الاستخدام (دقيقة واحدة)

1. **أدخل رمز السهم** في حقل "اسم/رمز الشركة"
   ```
   مثال: AAPL
   ```

2. **اضغط "إضافة"**

3. **اختر الفترة** (افتراضيًا: آخر 3 سنوات)

4. **اضغط "جلب البيانات"**

5. **انتظر** (3-5 ثوانٍ)

6. **راجع النتائج** في تبويب "التحليل الاستراتيجي"

---

## ✅ تحقق من النجاح

### افتح تبويب "التحليل الاستراتيجي" وتأكد من:

- ✅ **AR_Days** له قيمة (مثل: 45.23 days)
- ✅ **Retention_Ratio** له نسبة (مثل: 87.50%)
- ✅ **Beta** له قيمة (مثل: 1.23)
- ✅ **FCF_Yield** له نسبة (مثل: 3.45%)
- ✅ **Economic_Spread** له قيمة (مثل: 8.76%)

**إذا رأيت قيمًا حقيقية بدلاً من N/A → النظام يعمل! 🎉**

## 📁 تصدير النتائج

بعد جلب البيانات:

1. اضغط **"تصدير إلى Excel"**
2. اختر مكان الحفظ
3. افتح الملف في Excel

سيحتوي على 3 أوراق عمل:
- **Raw_by_Year**: البيانات الخام من SEC
- **Ratios**: النسب المالية
- **Strategic**: التحليل الاستراتيجي الكامل
