# 🎓 نظام التدريب المستمر للتحليل الذكي

## 🎯 نظرة عامة

تم إضافة **نظام تدريب مستمر (Continuous Learning)** يتعلم من كل تحليل ويحسّن دقة التنبؤات تلقائياً!

### **المشكلة السابقة:**
```
❌ النتائج ثابتة لجميع الشركات
❌ يستخدم قواعد ثابتة فقط
❌ لا يتعلم من البيانات
❌ نفس احتمالية الاحتيال لشركات مختلفة
```

### **الحل الجديد:**
```
✅ يتعلم من كل شركة تحللها
✅ يحسّن الدقة تلقائياً
✅ نتائج مخصصة لكل شركة
✅ تدريب مستمر على البيانات الجديدة
```

## 🤖 النماذج الخمسة

### **1️⃣ نموذج كشف الاحتيال**

**يتعلم:**
- متى تكون Accruals عالية **حقاً** (بناءً على البيانات)
- العلاقة بين Net Margin و OCF Margin
- الأنماط في Z-Score للشركات المشبوهة

**قبل التدريب:**
```python
if abs(accruals) > 0.08:  # قاعدة ثابتة
    fraud_score += 0.15
```

**بعد التدريب:**
```python
# يستخدم العتبة المتعلمة من البيانات
threshold = model['thresholds']['accruals_high']  # مثلاً: 0.12
if abs(accruals) > threshold:
    fraud_score += weight * (accruals / threshold)
```

**النتيجة:**
- دقة أعلى
- نتائج مختلفة للشركات المختلفة
- يتكيف مع كل قطاع

---

### **2️⃣ نموذج التنبؤ بالتعثر**

**يتعلم:**
- عتبات Z-Score الحقيقية للخطر
- متى يكون Debt/EBITDA خطيراً فعلاً
- أهمية كل عامل نسبياً

**التحسين:**
```
قبل: احتمالية ثابتة لكل Z-Score
بعد: احتمالية مبنية على بيانات آلاف الشركات
```

---

### **3️⃣ نموذج استدامة النمو**

**يتعلم:**
- العلاقة بين ROIC والنمو المستدام
- متى يشير نمو الديون إلى خطر
- توزيع الدرجات للشركات الناجحة

---

### **4️⃣ نموذج رأس المال العامل**

**يتعلم:**
- CCC الطبيعي لكل قطاع
- متى يؤدي النمو السريع لأزمة سيولة
- الاتجاهات الخطيرة في DSO و DIO

---

### **5️⃣ نموذج جودة الاستثمار**

**يتعلم:**
- الارتباط بين المقاييس المختلفة
- توزيع النتائج للشركات عالية الجودة
- أي العوامل أهم للقطاع المحدد

## 📈 تحسين الدقة

### **الصيغة المستخدمة:**
```python
# دمج نتائج ML مع القواعد
final_result = (ml_prediction * 0.7) + (rules_prediction * 0.3)
```

**لماذا 70% ML و 30% Rules؟**
- **70% ML:** يتعلم من البيانات الحقيقية
- **30% Rules:** يحافظ على الاستقرار والمنطق المالي

**مع مرور الوقت:**
```
بعد 50 شركة:  ML 70% | Rules 30%
بعد 100 شركة: ML 80% | Rules 20%
بعد 500 شركة: ML 90% | Rules 10%
```
